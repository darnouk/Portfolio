Catholic Religious Community Explorer

An interactive web map to explore Catholic religious communities across North America.



This web map was created using ArcGIS Online to provide an interactive way to explore Catholic religious communities and their locations.
The map features intuitive filtering and search capabilities to help users discover information about various religious orders.

How to use the map:
• Toggle visibility filters to show or hide different types of communities
• Use the "Filter by Order" dropdown to narrow results by specific religious orders
• Click on any point to view detailed information about that location, including the community name and website
• Use the search bar to find specific addresses or places

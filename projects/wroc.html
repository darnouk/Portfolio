WROC Project Tracker

Explore Wisconsin's orthoimagery and Lidar projects from 2010-2025



This comprehensive web mapping application tracks Wisconsin Regional Orthoimagery Consortium (WROC) orthoimagery and Lidar project data spanning 15 years of work.
I created this internal tool to help my company visualize and manage project locations, specifications, and timelines across the state of Wisconsin.

How to use the tracker:
• Select project type to view either orthoimagery or Lidar projects
• Choose year/cycle to filter projects by specific time periods
• Filter by resolution or quality level (QL) to narrow results by technical specifications
• Pick a location on the map to view detailed project information for that county or area

Key Features:
• Interactive filtering by multiple parameters (project type, year, resolution, location)
• Visual representation of project coverage across Wisconsin counties
• Comprehensive dataset covering 2010-2025 WROC projects
• Color-coded visualization for easy identification of different project types


Note: This map contains proprietary company data and is accessible through a private link only.
